template<typename B , typename KeyHash >
void Vector< B, KeyHash >::factorise_norm ( )
inline

Factorises the norm of the vector, i.e. performs ( v \to |v| \frac{v}{|v|} ). Since the overall norm is stored in a float type which cannot overflow, this prevents overflow of the components of the vector when using a standard float type for the values.
