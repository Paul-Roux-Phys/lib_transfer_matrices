\hypertarget{index_autotoc_md0}{}\doxysection{\texorpdfstring{Purpose of this library}{Purpose of this library}}\label{index_autotoc_md0}
This small template library offers a few convenient data types and methods designed to simplify writing efficient transfer matrix programs.

Specifically, the program is designed to perform computations on transfer matrices written in the form

\$\$T = \textbackslash{}prod R\+\_\+i,\$\$

where the $R_i$ are sparse matrices, in the sense that there exists a basis $|e_i \rangle$ such that $R |e_i \rangle$ has only a few non-\/zero components in the basis $|e_i\rangle$.\hypertarget{index_autotoc_md1}{}\doxysection{\texorpdfstring{Installation}{Installation}}\label{index_autotoc_md1}
This library is a template library, meaning the library itself requires no compilation. Simply clone or download the \href{https://github.com/Paul-Roux-Phys/lib_transfer_matrices/}{\texttt{ git repo}} to {\ttfamily some/path/lib\+\_\+transfter\+\_\+matrices} and include {\ttfamily \doxylink{_transfer_matrices_8hpp_source}{Transfer\+Matrices.\+hpp}} in your project\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{preprocessor}{\#include\ "{}TransferMatrices.hpp"{}}}

\end{DoxyCode}


Make sure to compile with the option {\ttfamily -\/I/path/to/lib\+\_\+transfer\+\_\+matrices/include} so that the compiler finds the file {\ttfamily \doxylink{_transfer_matrices_8hpp_source}{Transfer\+Matrices.\+hpp}}.\hypertarget{index_autotoc_md2}{}\doxysubsection{\texorpdfstring{Dependencies}{Dependencies}}\label{index_autotoc_md2}
